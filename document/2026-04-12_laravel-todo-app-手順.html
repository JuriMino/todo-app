Laravel ToDoアプリ 作成手順

作業日: 2026-04-12

1. Laravelプロジェクトの作成 (Docker/Sail)

Laravel公式のDocker環境「Sail」でプロジェクトを作成。

curl -s "https://laravel.build/todo-app?php=84" | bash
デフォルトのPHPバージョン(8.5)のイメージが存在しなかったため ?php=84 を指定。

Sailエイリアスの登録

echo "alias sail='[ -f sail ] && sh sail || sh vendor/bin/sail'" >> ~/.zshrc
source ~/.zshrc

起動

cd todo-app
sail up -d

2. テーブル設計

カラム型説明
idbigint主キー
titlestring(100)タイトル (必須)
descriptiontext詳細 (任意)
statusstring(20)未着手 / 進行中 / 完了
started_atdate開始予定日 (任意)
completed_atdate完了予定日 (任意)
created_at / updated_atdatetime自動
開始日/完了日は「予定」として手動入力。実際の開始日・完了日は将来的に別カラムで追加する方針。

3. モデル・マイグレーション・コントローラー作成

sail artisan make:model Todo -mcr

-mcr = migration + controller + resource

マイグレーションファイル編集

database/migrations/xxxx_xx_xx_create_todos_table.php

public function up(): void
{
    Schema::create('todos', function (Blueprint $table) {
        $table->id();
        $table->string('title', 100);
        $table->text('description')->nullable();
        $table->string('status', 20)->default('未着手');
        $table->date('started_at')->nullable();
        $table->date('completed_at')->nullable();
        $table->timestamps();
    });
}

マイグレーション実行

sail artisan migrate

Sequel Ace での接続確認

4. モデル設定

app/Models/Todo.php

protected $fillable = [
    'title',
    'description',
    'status',
    'started_at',
    'completed_at',
];

protected $casts = [
    'started_at' => 'date',
    'completed_at' => 'date',
];

5. ルーティング

routes/web.php

use App\Http\Controllers\TodoController;

Route::resource('todos', TodoController::class);

確認コマンド:

sail artisan route:list

6. コントローラー (index)

app/Http/Controllers/TodoController.php

use App\Models\Todo;

public function index()
{
    $todos = Todo::orderBy('created_at', 'desc')->get();

    return view('todos.index', compact('todos'));
}

8. 一覧ビュー

resources/views/todos/index.blade.php

@extends('layouts.app')

@section('title', 'ToDo一覧')

@section('content')
    <div class="flex justify-between items-center mb-6">
        <h2 class="text-xl font-semibold text-gray-700">ToDo一覧</h2>
        <a href="{{ route('todos.create') }}"
           class="bg-blue-600 hover:bg-blue-700 text-white px-4 py-2 rounded shadow">
            + 新規作成
        </a>
    </div>

    <div class="bg-white shadow rounded-lg overflow-hidden">
        <table class="min-w-full divide-y divide-gray-200">
            <thead class="bg-gray-100">
                <tr>
                    <th class="px-6 py-3 text-left text-sm font-medium text-gray-600">タイトル</th>
                    <th class="px-6 py-3 text-left text-sm font-medium text-gray-600">ステータス</th>
                    <th class="px-6 py-3 text-left text-sm font-medium text-gray-600">開始日</th>
                    <th class="px-6 py-3 text-left text-sm font-medium text-gray-600">完了日</th>
                    <th class="px-6 py-3 text-left text-sm font-medium text-gray-600">作成日</th>
                </tr>
            </thead>
            <tbody class="divide-y divide-gray-200">
                @forelse ($todos as $todo)
                    <tr class="hover:bg-gray-50">
                        <td class="px-6 py-4 text-gray-800">{{ $todo->title }}</td>
                        <td class="px-6 py-4">
                            <span @class([
                                'px-2 py-1 rounded text-xs font-semibold',
                                'bg-gray-200 text-gray-700' => $todo->status === '未着手',
                                'bg-yellow-200 text-yellow-800' => $todo->status === '進行中',
                                'bg-green-200 text-green-800' => $todo->status === '完了',
                            ])>
                                {{ $todo->status }}
                            </span>
                        </td>
                        <td class="px-6 py-4 text-gray-600">{{ $todo->started_at?->format('Y-m-d') ?? '-' }}</td>
                        <td class="px-6 py-4 text-gray-600">{{ $todo->completed_at?->format('Y-m-d') ?? '-' }}</td>
                        <td class="px-6 py-4 text-gray-600">{{ $todo->created_at->format('Y-m-d') }}</td>
                    </tr>
                @empty
                    <tr>
                        <td colspan="5" class="px-6 py-8 text-center text-gray-500">
                            ToDoがまだありません
                        </td>
                    </tr>
                @endforelse
            </tbody>
        </table>
    </div>
@endsection
