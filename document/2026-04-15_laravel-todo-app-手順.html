Laravel ToDoアプリ 作成手順 (Day 4)

作業日: 2026-04-15 / 内容: ToDo削除機能の実装

本日のゴール

  • ToDoの削除機能を実装
  • 削除前の確認ダイアログでうっかり削除を防止
  • 一覧画面に成功メッセージを表示

処理の流れ

[詳細画面] ↓ 「削除」ボタンクリック ↓ ブラウザの confirm() ダイアログ ↓ OK ↓ POST /todos/{id} (_method=DELETE) [TodoController@destroy] ↓ ルートモデルバインディングで該当レコード取得 ↓ $todo->delete() [DB: レコード削除] ↓ redirect [一覧画面] + 成功メッセージ「ToDoを削除しました」

1. コントローラーの destroy メソッド

app/Http/Controllers/TodoController.php

public function destroy(Todo $todo)
{
    $todo->delete();

    return redirect()->route('todos.index')
        ->with('success', 'ToDoを削除しました');
}
ポイント:
  • ルートモデルバインディング (Todo $todo) で対象レコードを自動取得
  • ->with('success', ...) でフラッシュメッセージをセッションに保存
  • ルートは Route::resource('todos', ...) で既に登録済み

2. 詳細画面に削除ボタンを追加

resources/views/todos/show.blade.php のアクションエリア:

<div class="px-8 py-4 bg-slate-50 border-t border-slate-100 flex justify-between items-center">
    <form action="{{ route('todos.destroy', $todo) }}" method="POST"
          onsubmit="return confirm('このToDoを削除します。よろしいですか？');">
        @csrf
        @method('DELETE')
        <button type="submit"
                class="px-4 py-2 text-sm font-medium text-red-600 hover:bg-red-50 rounded-lg transition">
            削除
        </button>
    </form>

    <a href="{{ route('todos.edit', $todo) }}"
       class="px-4 py-2 text-sm font-medium bg-slate-900 hover:bg-slate-800 text-white rounded-lg transition">
        編集
    </a>
</div>

ポイント解説

なぜ <a> ではなく <form> なのか

削除は DELETEメソッド で送信する必要がある。<a> タグは GET しか送れないため、<form> + @method('DELETE') を使う。

@method('DELETE') の仕組み

HTMLフォームは本来 GET / POST しか送れない ↓ Laravel は hidden フィールド _method=DELETE を自動で埋め込む ↓ <input type="hidden" name="_method" value="DELETE"> ↓ サーバー側で Laravel が DELETE リクエストとして解釈し、 destroy メソッドへルーティング

@csrf とは

Cross-Site Request Forgery (CSRF) 対策のトークン。Laravelでは POST/PUT/DELETE すべてで必須。省くと 419 エラー。

onsubmit の確認ダイアログ

onsubmit="return confirm('このToDoを削除します。よろしいですか？');"
  • ブラウザ標準の confirm() でOK/キャンセルを選択できる
  • キャンセル時は false が返り、フォーム送信がキャンセルされる
  • シンプルだが実用十分。モダンなモーダルにしたい場合は JS で自作も可

動作確認

  1. 詳細画面で「削除」ボタンをクリック
  2. 「このToDoを削除します。よろしいですか？」の確認ダイアログが表示される
  3. OKで削除 → 一覧画面に戻り、削除成功メッセージ表示
  4. キャンセルで何も起きない
